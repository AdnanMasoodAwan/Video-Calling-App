\doxysubsection{Detailed Description}
The channel media options.

Agora supports publishing multiple audio streams and one video stream at the same time and in the same \mbox{\hyperlink{structagora_1_1rtc_1_1_rtc_connection}{Rtc\+Connection}}. For example, {\ttfamily publish\+Audio\+Track}, {\ttfamily publish\+Custom\+Audio\+Track} and {\ttfamily publish\+Media\+Player\+Audio\+Track} can be true at the same time; but only one of {\ttfamily publish\+Camera\+Track}, {\ttfamily publish\+Screen\+Track}, {\ttfamily publish\+Custom\+Video\+Track}, and {\ttfamily publish\+Encoded\+Video\+Track} can be true at the same time.
