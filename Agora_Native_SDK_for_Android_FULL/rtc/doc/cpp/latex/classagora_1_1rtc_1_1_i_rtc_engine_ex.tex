{\footnotesize\ttfamily virtual int agora\+::rtc\+::\+I\+Rtc\+Engine\+Ex\+::join\+Channel\+Ex (\begin{DoxyParamCaption}\item[{const char $\ast$}]{token,  }\item[{const \mbox{\hyperlink{structagora_1_1rtc_1_1_rtc_connection}{Rtc\+Connection}} \&}]{connection,  }\item[{const \mbox{\hyperlink{structagora_1_1rtc_1_1_channel_media_options}{Channel\+Media\+Options}} \&}]{options,  }\item[{\mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtc_engine_event_handler}{I\+Rtc\+Engine\+Event\+Handler}} $\ast$}]{event\+Handler }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [pure virtual]}}

Joins a channel with media options.

This method enables users to join a channel. Users in the same channel can talk to each other, and multiple users in the same channel can start a group chat. Users with different App I\+Ds cannot call each other.

A successful call of this method triggers the following callbacks\+:
\begin{DoxyItemize}
\item The local client\+: The {\ttfamily on\+Join\+Channel\+Success} and {\ttfamily on\+Connection\+State\+Changed} callbacks.
\item The remote client\+: {\ttfamily on\+User\+Joined}, if the user joining the channel is in the Communication profile or is a host in the Live-\/broadcasting profile.
\end{DoxyItemize}

When the connection between the client and Agora\textquotesingle{}s server is interrupted due to poor network conditions, the S\+DK tries reconnecting to the server. When the local client successfully rejoins the channel, the S\+DK triggers the {\ttfamily on\+Rejoin\+Channel\+Success} callback on the local client.

Compared to {\ttfamily join\+Channel}, this method adds the options parameter to configure whether to automatically subscribe to all remote audio and video streams in the channel when the user joins the channel. By default, the user subscribes to the audio and video streams of all the other users in the channel, giving rise to usage and billings. To unsubscribe, set the {\ttfamily options} parameter or call the {\ttfamily mute} methods accordingly.

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item This method allows users to join only one channel at a time.
\item Ensure that the app ID you use to generate the token is the same app ID that you pass in the {\ttfamily initialize} method; otherwise, you may fail to join the channel by token.
\end{DoxyItemize}
\end{DoxyNote}

Agora supports adding only one watermark image onto the local video, and the newly watermark image replaces the previous one.

The watermark position depends on the settings in the \mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtc_engine_a9bcbdcee0b5c52f96b32baec1922cf2e}{set\+Video\+Encoder\+Configuration}} method\+:
