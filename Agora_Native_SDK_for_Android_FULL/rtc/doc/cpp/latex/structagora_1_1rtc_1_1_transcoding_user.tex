\doxysubsubsection{\texorpdfstring{audioChannel}{audioChannel}}
{\footnotesize\ttfamily int agora\+::rtc\+::\+Transcoding\+User\+::audio\+Channel}

The audio channel used by the host\textquotesingle{}s audio in the output audio. The default value is 0, and the value range is \mbox{[}0, 5\mbox{]}.
\begin{DoxyItemize}
\item {\ttfamily 0}\+: (Recommended) The defaut setting, which supports dual channels at most and depends on the upstream of the host.
\item {\ttfamily 1}\+: The host\textquotesingle{}s audio uses the FL audio channel. If the host\textquotesingle{}s upstream uses multiple audio channels, the Agora server mixes them into mono first.
\item {\ttfamily 2}\+: The host\textquotesingle{}s audio uses the FC audio channel. If the host\textquotesingle{}s upstream uses multiple audio channels, the Agora server mixes them into mono first.
\item {\ttfamily 3}\+: The host\textquotesingle{}s audio uses the FR audio channel. If the host\textquotesingle{}s upstream uses multiple audio channels, the Agora server mixes them into mono first.
\item {\ttfamily 4}\+: The host\textquotesingle{}s audio uses the BL audio channel. If the host\textquotesingle{}s upstream uses multiple audio channels, the Agora server mixes them into mono first.
\item {\ttfamily 5}\+: The host\textquotesingle{}s audio uses the BR audio channel. If the host\textquotesingle{}s upstream uses multiple audio channels, the Agora server mixes them into mono first.
\item {\ttfamily 0x\+FF} or a value greater than 5\+: The host\textquotesingle{}s audio is muted, and the Agora server removes the host\textquotesingle{}s audio.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
If the value is not {\ttfamily 0}, a special player is required. 
\end{DoxyNote}
