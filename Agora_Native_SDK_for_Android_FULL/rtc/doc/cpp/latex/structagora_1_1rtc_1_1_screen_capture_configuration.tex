(For Windows and mac\+OS only) The relative position of the shared region to the whole screen. For details, see \mbox{\hyperlink{structagora_1_1rtc_1_1_rectangle}{Rectangle}}.

If you do not set this parameter, the S\+DK shares the whole screen. If the region you set exceeds the boundary of the screen, only the region within in the screen is shared. If you set width or height in \mbox{\hyperlink{structagora_1_1rtc_1_1_rectangle}{Rectangle}} as 0, the whole screen is shared.
