\doxysubsection{Detailed Description}
{\ttfamily \mbox{\hyperlink{classagora_1_1rtc_1_1_i_local_audio_track}{I\+Local\+Audio\+Track}}} is the basic class for local audio tracks, providing main methods of local audio tracks.

You can create a local audio track by calling one of the following methods\+:
\begin{DoxyItemize}
\item {\ttfamily create\+Local\+Audio\+Track}
\item {\ttfamily create\+Custom\+Audio\+Track}
\item {\ttfamily create\+Media\+Player\+Audio\+Track} You can also use the A\+P\+Is in the \mbox{\hyperlink{classagora_1_1rtc_1_1_i_n_g_audio_device_manager}{I\+Audio\+Device\+Manager}} class if multiple recording devices are available in the system.
\end{DoxyItemize}

After creating local audio tracks, you can publish one or more local audio tracks by calling \mbox{\hyperlink{classagora_1_1rtc_1_1_i_local_user_a5e5cc962971f99f07a692d82ac02ff1d}{publish\+Audio}}.
