\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classio_1_1agora_1_1rtm_1_1_rtm_client}{Rtm\+Client}} class.

This class provides the main methods that can be invoked by your app.

\mbox{\hyperlink{classio_1_1agora_1_1rtm_1_1_rtm_client}{Rtm\+Client}} is the basic interface class of the Agora R\+TM S\+DK. Creating an \mbox{\hyperlink{classio_1_1agora_1_1rtm_1_1_rtm_client}{Rtm\+Client}} object and then calling the methods of this object enables you to use Agora R\+TM S\+DK\textquotesingle{}s functionality.
